Présentation du Résultat

Réalisation

1. Pour commencer, vous avez besoin de télecharger les deux fichiers qui vous serons nécessaire a la Réalisation du tutoriel.

2. Décompresser le fichier pour obtenier le document "x64" sur votre bureau.

3. Rendez-vous dans votre dossier racine de Grand Theft Auto V (exemple: C:\Game\Grand Theft Auto V).

4. Pour finir, faite un glisser-déposer de "x64" de votre bureau dans la racine de GTA V.

4.1. 2 Fichiers vont etre remplacer durant le glisser-déposer (RESIDENT.rpf | WEAPONS_PLAYER.rpf).

! Attention ! Si vous lancer GTAV par le launcher Rockstar pour jouer en solo, il est possible que le launcher lance une mise a jour pour remplacer les deux fichiers!
Il faudra donc refaire toute la manipulation.
